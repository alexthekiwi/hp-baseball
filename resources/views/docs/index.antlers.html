Getting Started

Logging Into the CMS

  1. Navigate to /admin (e.g., hpbaseball.co.nz/admin)
  2. Enter your email and password
  3. Click "Log in" to access the dashboard
{{ partial:docs/screenshot path="docs/login-screen.png" class="my-4" }}

Tip: Bookmark the login page for quick access

Understanding the Dashboard

The dashboard is your control center. Key areas include:

  • Collections: Where all your content lives (Articles, Pages, etc.)
  • Assets: Your media library for images and files
  • Navigation: Menu structure management
  • Globals: Site-wide settings

{{# Pages #}}

Pages

Pages are the main sections of your website (About, Contact, etc.)

Creating a New Page:

  1. Go to Collections → Pages
  2. Click "Create Entry"
  3. Fill in the required fields:
    • Title: Page name (shows in browser tab)
    • Nav Title: Shorter name for menus (optional)
    • Content: Use the page builder blocks
  4. Set the URL slug in the sidebar
  5. Click "Save & Publish"
{{ partial:docs/screenshot path="docs/page-editor.png" class="my-4" }}

Using the Block Page Builder:

Pages use a flexible block-based system that allows you to piece together content using predefined blocks. This gives you complete control over page layout and structure.

Available Block Types:
  • Hero: Large banner sections with images and call-to-action buttons
  • Content Block: Text content with rich formatting
  • Image Gallery: Display multiple images in a grid
  • Testimonials: Customer reviews and quotes
  • Featured Articles: Highlight recent news or articles
  • Contact Forms: Add contact forms to pages
  • And many more...
Adding Blocks:
  1. In the page editor, scroll to the "Page Builder" section
  2. Click "Add Block" to see available block types
  3. Select the block type you want to add
  4. Configure the block's content and settings
  5. Drag blocks to reorder them as needed
  6. Use the collapse/expand arrows to organize your blocks
{{ partial:docs/screenshot path="docs/page-builder-1.png" class="my-4" }} {{ partial:docs/screenshot path="docs/page-builder-2.png" class="my-4" }}

Need a New Block Type? If you need a block type that doesn't exist, contact Alex Clark at hello@alexclark.co.nz for custom development.

{{# Articles #}}

Articles (News & Updates)

Articles are for news, announcements, and blog posts.

Publishing an Article:

  1. Go to Collections → Articles
  2. Click "Create Entry"
  3. Add your content:
    • Title: Article headline
    • Author: Select from users
    • Excerpt: Brief summary (shows in listings)
    • Content: Full article text
    • Date: Publication date
    • Tags: Categories for organization
  4. Save & Publish when ready
{{ partial:docs/screenshot path="docs/article-editor.png" class="my-4" }}
{{# Products #}}

Products (Merchandise)

Add club merchandise and items for sale.

Adding a Product:

  1. Go to Collections → Products
  2. Click "Create Entry"
  3. Add product information:
    • Title: Product name
    • Thumbnail: Main product image
    • Images: Additional photos
    • Content: Description and details
    • Price: Cost in dollars
  4. Save & Publish
{{ partial:docs/screenshot path="docs/product-editor.png" class="my-4" }}
{{# Sponsors #}}

Sponsors

Manage club sponsors and partners.

Adding a Sponsor:

  1. Go to Collections → Sponsors
  2. Click "Create Entry"
  3. Enter sponsor details:
    • Title: Sponsor name
    • Logo: Upload sponsor logo
    • Website: Link to sponsor's site
  4. Save to add to sponsor section

Note: Sponsors automatically appear on the homepage

{{# Forms #}}

Forms

Manage contact forms and other data collection forms. Forms allow visitors to submit information which you can receive via email and view in the admin panel.

Viewing Form Submissions:

  1. Go to {{ partial:docs/link url="/admin/forms/contact" text="Forms → Contact" }} (or your specific form)
  2. View all submissions in a table format
  3. Click on individual submissions to see details
  4. Export submissions as CSV if needed
{{ partial:docs/screenshot path="docs/form-submissions.png" class="my-4" }}

Configuring Form Settings:

To configure email notifications and other form settings:

  1. Go to {{ partial:docs/link url="/admin/forms/contact" text="Forms → Contact" }}
  2. Click on the form name to access settings
  3. Configure email settings:
    • Email Recipients: Add email addresses to receive notifications
    • Email Subject: Set the subject line for notifications
    • HTML/Text View: Always choose "form_sender" for emails going to admin and "form_owner" for emails going to the site user
  4. Save your changes
{{ partial:docs/screenshot path="docs/form-config.png" class="my-4" }}

Editing Form Fields:

To modify the fields that appear on your form:

  1. Go to the form settings (as above)
  2. Click on "Blueprint" or "Fields" tab
  3. Add, remove, or modify form fields:
    • Text Fields: Name, email, phone number
    • Textarea: For longer messages
    • Select/Dropdown: Multiple choice options
    • Assets: Allow file attachments
    • Validation: Set required fields and validation rules
  4. Save and test your form
{{ partial:docs/screenshot path="docs/form-blueprint.png" class="my-4" }}

Adding Forms to Pages:

To add a form to any page using the block editor:

  1. Edit the page where you want the form
  2. In the Page Builder, click "Add Block"
  3. Select the "Form" block type
  4. Choose which form to display (e.g., Contact Form)
  5. Configure block settings like title and styling
  6. Save and publish the page

Pro Tip: Test your form after making changes to ensure emails are being delivered and all fields work correctly.

{{# Redirects #}}

Redirects

Set up URL redirects for old or moved pages.

Creating a Redirect:

  1. Go to Collections → Redirects
  2. Click "Create Entry"
  3. Configure redirect:
    • Old URL: The URL to redirect from (e.g., /old-page)
    • Response Code: 301 for permanent, 302 for temporary
    • Redirect Type: Choose Entry or URL
    • Destination: Select page or enter URL
  4. Save to activate redirect

{{# Common Tasks Section #}}

Common Tasks

Uploading Images

  1. Click the image field in any content editor
  2. Choose "Upload" or select from existing assets
  3. Drag and drop or browse for your image
  4. Add alt text for accessibility
  5. Click "Save" to insert
{{ partial:docs/screenshot path="docs/image-upload.png" class="my-4" }}

Best Practice: Resize images before uploading. Aim for under 2MB per image.

Using the Content Editor (Bard)

The Bard editor offers rich text formatting:

  • Headers: Use H2 and H3 for sections
  • Bold/Italic: Highlight important text
  • Lists: Create bullet or numbered lists
  • Links: Add internal or external links
{{ partial:docs/screenshot path="docs/bard-editor.png" class="my-4" }}

SEO Settings

Both pages and articles have an SEO tab where you can optimize for search engines:

  • Meta Title: Page title for search results (50-60 characters)
  • Meta Description: Brief page summary (150-160 characters)
  • Social Images: Image for social media sharing
  • No Index: Check this to hide the page from search engine crawlers
  • URL Slug: Clean, descriptive URL (found in the sidebar)
{{ partial:docs/screenshot path="docs/seo-settings.png" class="my-4" }}

Publishing Options

Control when and how content goes live:

  • Save & Publish: Use the "Published" toggle to control visibility
  • Published Toggle: Turn on to make content live, turn off to hide it
  • Future Articles: Articles with dates set in the future won't appear on the site until that date
  • Draft Mode: Leave "Published" off to save drafts without making them live

Tip: Use future dates for articles to schedule automatic publishing of news and announcements.

{{ partial:docs/screenshot path="docs/publish-options.png" class="my-4 max-w-sm" }}

Enable/Disable Shop

Control whether customers can add items to their cart and make purchases on the site.

How to Enable or Disable the Shop:

  1. Go to {{ partial:docs/link url="/admin/globals/settings" text="Globals → Settings" }}
  2. Find the "Shop Enabled" toggle switch
  3. Toggle it:
    • ON: Shop is active - customers can add items to cart and checkout
    • OFF: Shop is disabled - displays a friendly message to customers
  4. Click "Save" to apply the changes
{{ partial:docs/screenshot path="docs/enable-shop.png" class="my-4" }}

Important: When the shop is disabled, customers will see a friendly message explaining that the shop is temporarily unavailable. Any items already in their cart will be preserved for when the shop is re-enabled.

When to Disable the Shop:

  • During maintenance or updates to product pricing
  • When inventory is being updated or restocked
  • During off-season periods
  • When payment processing needs to be temporarily suspended

{{# Tips & Best Practices Section #}}

Tips & Best Practices

Quick Tips

Image Guidelines

  • Hero images: 1920x1080px recommended
  • Thumbnails: 800x600px minimum
  • Logos: PNG with transparent background
  • File size: Keep under 2MB when possible

Content Organization

  • Use clear, descriptive titles
  • Keep URLs short and meaningful
  • Tag articles for easy filtering
  • Set parent pages for proper hierarchy

When to Use Each Content Type

  • Pages: Permanent content (About, Contact)
  • Articles: Time-sensitive news and updates
  • Fixed Blocks: Reusable content sections
  • Products: Club merchandise and items for sale
  • Sponsors: Partner organizations and supporters

Before Publishing

  • Preview your content first
  • Check all links work correctly
  • Verify images display properly
  • Review on mobile and desktop
